Projeto TOO - Sistema de Adoção de Pets

Descrição:

  Este projeto foi desenvolvido para a disciplina de Tecnologia de Orientação a Objetos (TOO) e tem como objetivo simular um sistema de gerenciamento de adoção de pets.
  O sistema permite o cadastro e gerenciamento de animais e adotantes, aplicando conceitos de Programação Orientada a Objetos e padrões de projeto estudados durante a disciplina.

Objetivo:

  Desenvolver um sistema orientado a objetos capaz de auxiliar no controle de processos de adoção de animais, promovendo a organização das informações de pets disponíveis e de seus possíveis adotantes.

Funcionalidades Implementadas:
  Atualmente o sistema possui:

  Classe Animal para representar os animais cadastrados;
  Classe Adotante para representar os interessados na adoção;
  Estrutura inicial para evolução do sistema.
